# eros

## requirements
* Docker
* KinD
* Just

## Get started

Start the API Backend:
```shell
eros server start
```

Start the Frontend Server:
```shell
eros frontend start
```

## Documentation
### Backend API
I am using the Swagger Tooling to generate the needed API documentations.
You can access the documentation with:
```shell
http://localhost:3000/swagger/index.html
```
